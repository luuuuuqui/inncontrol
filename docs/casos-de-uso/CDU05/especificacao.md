# CDU05 – Atualizar Quarto

**Descrição:** Permite que o administrador atualize as informações de um quarto já cadastrado no sistema.

**Ator Primário:** Administrador.

**Pré-condições:** 


Deve existir ao menos um quarto cadastrado.

**Pós-condições:** Os atributos do quarto estarão atualizados.

## Fluxo Principal

1. O administrador abre a aba "Atualizar Quarto".

2. O administrador escolhe o quarto no menu drop-down.

3. O sistema exibe todos os atributos do quarto sendo editáveis.

4. O administrador edita os atributos
